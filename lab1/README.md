🧠 Опис проєкту

Цей репозиторій містить просту консольну програму, написану мовою Java, яка демонструє роботу з аргументами командного рядка.

Програма приймає аргументи під час запуску та виводить їх на екран разом із загальною кількістю.

🎯 Мета роботи
- Ознайомлення з методом main(String[] args)
- Розуміння передачі аргументів у Java
- Практика роботи з масивами
- Вивчення базової структури Java-програми
🛠 Використані технології
- Java
- Консольний інтерфейс
- Командний рядок
⚙️ Як запустити
- Скомпілювати програму:
javac main.java
- Запустити з аргументами:
java main hello world 123
📌 Результат роботи

Програма:
- перевіряє наявність аргументів
- виводить кожен аргумент окремо
- підраховує їх кількість
